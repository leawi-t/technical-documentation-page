Introduction
What is Java?

Java is a popular, object-oriented programming language and a software platform known for its portability across different devices through the "write once, run anywhere" principle.

It powers billions of devices, including Android operating systems and applications, enterprise software, and dynamic web applications, making it a versatile choice for various development needs.

Java contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core Java can be extended for a variety of purposes by supplementing it with additional objects.

  • Easy to learn
  • Platform independent
  • Robust and secure
  • Versatile
  • Scalable

Variables

A variable is a symbolic name that represents a storage location in the computer's memory, used to hold a value or data that can change during the program's execution.

Variables are fundamental for performing calculations, making decisions, and making programs dynamic by allowing data to be stored and manipulated as needed.

Data Types

Data types in programming classify the kind of values a variable can hold and define the operations that can be performed on those values. Understanding data types is fundamental for writing correct and efficient code.

if-else statement

An if-else statement is a programming construct that executes different blocks of code based on whether a condition is true or false.

Condition Evaluation: The program first evaluates a condition provided within the if statement.

True Condition: If true, the code block associated with the if is executed.

False Condition: If false, the else block (if present) is executed instead.

// code to check whether number is even or odd public static void main{ int x=5; if(x%2==0) System.out.println("the number is even"); else if(x%2!=0) System.out.println("the number is odd"); else System.out.println("It is neither"); }

You may also chain multiple conditions using else if:

if (condition_1) { statement_1; } else if (condition_2) { statement_2; } else { statement_last; }
Loops

In Java, loops are control flow statements used to repeatedly execute a block of code based on a condition. Java offers three primary types: for, while, do-while, plus the enhanced for loop.

// code to print numbers from 1-10 for(int i=1; i<=10; i++){ System.out.println(i); }
OOP
What is OOP

Object-Oriented Programming (OOP) is a programming paradigm that structures software around objects, which are instances of classes containing both data (attributes) and functions (methods) that operate on that data.

This approach models real-world entities, making code more organized, reusable, and maintainable.

OOP Principles

Key principles of OOP include abstraction, encapsulation, inheritance, and polymorphism, which together allow for the creation of flexible, modular, and scalable software systems.

Core Components

Objects

The fundamental building blocks of an OOP system, representing real-world entities or abstract concepts. They have specific data (attributes) and behaviors (methods).

Classes

Blueprints or templates for creating objects. They define the structure and behavior that all objects of that class will have.

Attributes (Properties)

Variables that store the data or characteristics of an object.

Methods (Functions)

Functions defined within a class that perform actions or operations on an object's data.

Pillars Of OOP
Abstraction Hiding complex implementation details and showing only the essential features of an object. This simplifies the use of objects by providing a clean and understandable interface.
Encapsulation Bundling data and methods that operate on the data into a single unit, called an object. This hides the internal details of an object, protecting its data from unauthorized access.
Inheritance A mechanism that allows a new class (child class) to inherit properties and behaviors from an existing class (parent class). This promotes code reuse and creates hierarchical relationships between classes.
Polymorphism The ability of objects to take on many forms. Different objects can respond to the same method call in their own way, leading to more flexible and adaptable code.
